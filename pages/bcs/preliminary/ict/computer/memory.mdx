# Memory (মেমোরি)

## Key Topics (প্রধান আলোচ্য বিষয়)

1. **Primary Memory (প্রাথমিক মেমোরি)**

   - RAM (র‍্যাম)
     - Static RAM
     - Dynamic RAM
   - ROM (রম)
     - PROM
     - EPROM
     - EEPROM

2. **Secondary Memory (দ্বিতীয়ক মেমোরি)**

   - Hard Disk (হার্ড ডিস্ক)
   - SSD (এসএসডি)
   - Optical Storage (অপটিক্যাল স্টোরেজ)
   - Flash Memory (ফ্ল্যাশ মেমোরি)

3. **Cache Memory (ক্যাশে মেমোরি)**

   - L1 Cache
   - L2 Cache
   - L3 Cache

4. **Virtual Memory (ভার্চুয়াল মেমোরি)**
   - Paging (পেজিং)
   - Swapping (সোয়াপিং)
