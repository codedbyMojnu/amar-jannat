# Cognate Object: "dreamt a dream"

**প্রশ্ন:** I dreamt a wonderful dream. (এখানে 'a wonderful dream' কোন ধরনের object?)

(a) Retained object
(b) Cognate object
(c) Reflexive object
(d) Factive object

**সঠিক উত্তর:** (b) Cognate object

---

### **ব্যাখ্যা**

এই বাক্যে `a wonderful dream` হলো একটি Cognate Object। Cognate Object হলো এমন একটি object যা তার verb-এর সাথে একই মূল (root) বা ধাতু থেকে উদ্ভূত এবং অর্থের দিক থেকে সম্পর্কিত।

#### **Key Concepts:**

1.  **Cognate Object (সমধাতুজ কর্ম):**

    - সাধারণত, Intransitive Verb (যে verb-এর object লাগে না) object গ্রহণ করে না। যেমন: `dream`, `live`, `die`, `sing`, `run`।
    - কিন্তু যখন এই ধরনের verb-এর পরে এমন একটি object বসে যা verb-টির সাথে একই ধাতু বা উৎস থেকে তৈরি, তখন সেই object-কে Cognate Object বলে।
    - এই object-এর আগে সাধারণত একটি adjective থাকে।

    **Structure:** `Subject + Intransitive Verb + (Adjective) + Cognate Object`

#### **বিশ্লেষণ:**

- **Verb:** `dreamt` (from `dream`)
- **Object:** `dream`

এখানে verb এবং object উভয়ই `dream` মূল থেকে এসেছে। তাই `a wonderful dream` একটি Cognate Object।

#### **অন্যান্য উদাহরণ:**

- He **lived** a happy **life**. (Verb: `live`, Object: `life`)
- She **sang** a sweet **song**. (Verb: `sing`, Object: `song`)
- They **fought** a great **fight**. (Verb: `fight`, Object: `fight`)
- He **ran** a **race**. (Verb: `run`, Object: `race`)

#### **কেন অন্য অপশনগুলো ভুল?**

- **(a) Retained object**: Passive voice-এ ব্যবহৃত হয়। যখন ditransitive verb-কে passive করা হয়, তখন যে object-টি subject হয় না এবং verb-এর পরে থেকে যায়, তাকে Retained Object বলে। (e.g., I was given **a book**. এখানে `a book` হলো retained object)।
- **(c) Reflexive object**: যখন object এবং subject একই ব্যক্তি বা বস্তুকে বোঝায় এবং object হিসেবে reflexive pronoun (`myself`, `himself`, `themselves` ইত্যাদি) বসে। (e.g., He hurt **himself**.)
- **(d) Factive object**: `make`, `elect`, `select`, `nominate`, `call` ইত্যাদি verb-এর object-কে Factive Object বলে, যার অর্থ সম্পূর্ণ করার জন্য একটি object complement লাগে। (e.g., We made him **captain**. এখানে `him` হলো factive object এবং `captain` হলো complement)।

  (যেমন `dream` ও `dream`, `sing` ও `song`), তাহলে সেটি Cognate Object।
