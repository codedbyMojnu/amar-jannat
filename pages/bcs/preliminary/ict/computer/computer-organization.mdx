# Computer Organization and Components (কম্পিউটারের অংশগঠন)

## Key Topics (প্রধান আলোচ্য বিষয়)

1. **Central Processing Unit - CPU (কেন্দ্রীয় প্রক্রিয়াকরণ ইউনিট)**

   - Control Unit (কন্ট্রোল ইউনিট)
   - ALU (অঙ্কগণিত যুক্তি ইউনিট)
   - Registers (রেজিস্টার)

2. **Memory Unit (মেমোরি ইউনিট)**

   - Primary Memory (প্রাথমিক মেমোরি)
   - Secondary Memory (দ্বিতীয় মেমোরি)
   - Cache Memory (ক্যাশে মেমোরি)

3. **Input/Output Units (ইনপুট/আউটপুট ইউনিট)**

   - Input Devices (ইনপুট ডিভাইস)
   - Output Devices (আউটপুট ডিভাইস)
   - I/O Controllers (I/O কন্ট্রোলার)

4. **System Bus (সিস্টেম বাস)**
   - Data Bus (ডাটা বাস)
   - Address Bus (অ্যাড্রেস বাস)
   - Control Bus (কন্ট্রোল বাস)
