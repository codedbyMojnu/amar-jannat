# Computer Peripheral Devices (কম্পিউটারের পেরিফেরাল ডিভাইস)

## Key Topics (প্রধান আলোচ্য বিষয়)

1. **Input Devices (ইনপুট ডিভাইস)**

   - Keyboard (কীবোর্ড)
   - Mouse (মাউস)
   - Scanner (স্ক্যানার)
   - Webcam (ওয়েবক্যাম)
   - Microphone (মাইক্রোফোন)

2. **Output Devices (আউটপুট ডিভাইস)**

   - Monitor (মনিটর)
   - Printer (প্রিন্টার)
   - Speaker (স্পিকার)
   - Projector (প্রজেক্টর)

3. **Storage Devices (স্টোরেজ ডিভাইস)**

   - Hard Disk Drive (হার্ড ডিস্ক ড্রাইভ)
   - Solid State Drive (সলিড স্টেট ড্রাইভ)
   - USB Drive (ইউএসবি ড্রাইভ)
   - Optical Drives (অপটিক্যাল ড্রাইভ)

4. **Communication Devices (কমিউনিকেশন ডিভাইস)**
   - Modem (মডেম)
   - Network Card (নেটওয়ার্ক কার্ড)
   - Bluetooth Devices (ব্লুটুথ ডিভাইস)
