# Number Systems and Logic Gates (কম্পিউটারের নম্বর সিস্টেম ও লজিক গেট)

## Key Topics (প্রধান আলোচ্য বিষয়)

1. **Number Systems (নম্বর সিস্টেম)**

   - Binary (বাইনারি)
   - Decimal (ডেসিমাল)
   - Octal (অক্টাল)
   - Hexadecimal (হেক্সাডেসিমাল)
   - Number System Conversion (নম্বর সিস্টেম রূপান্তর)

2. **Basic Logic Gates (মৌলিক লজিক গেট)**

   - AND Gate (এন্ড গেট)
   - OR Gate (অর গেট)
   - NOT Gate (নট গেট)
   - NAND Gate (ন্যান্ড গেট)
   - NOR Gate (নর গেট)
   - XOR Gate (এক্সর গেট)
   - XNOR Gate (এক্সনর গেট)

3. **Boolean Algebra (বুলিয়ান অ্যালজেবরা)**
   - Boolean Operations (বুলিয়ান অপারেশন)
   - Truth Tables (ট্রুথ টেবিল)
   - Logic Expressions (লজিক এক্সপ্রেশন)
