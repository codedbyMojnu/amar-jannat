# Mobile Technology (সেলুলার ডাটা নেটওয়ার্ক বা মোবাইল প্রযুক্তি)

## Key Topics (প্রধান আলোচ্য বিষয়)

1. **Mobile Network Generations (মোবাইল নেটওয়ার্ক জেনারেশন)**

   - 1G (ফার্স্ট জেনারেশন)
   - 2G (সেকেন্ড জেনারেশন)
   - 3G (থার্ড জেনারেশন)
   - 4G (ফোর্থ জেনারেশন)
   - 5G (ফিফথ জেনারেশন)

2. **Cellular Network Components (সেলুলার নেটওয়ার্ক উপাদান)**

   - Base Stations (বেস স্টেশন)
   - Mobile Switching Centers (মোবাইল সুইচিং সেন্টার)
   - Cell Towers (সেল টাওয়ার)

3. **Mobile Technologies (মোবাইল প্রযুক্তি)**

   - GSM (জিএসএম)
   - CDMA (সিডিএমএ)
   - LTE (এলটিই)
   - VoLTE (ভিওএলটিই)

4. **Mobile Services (মোবাইল সেবা)**
   - Voice Calls (ভয়েস কল)
   - SMS (এসএমএস)
   - Mobile Internet (মোবাইল ইন্টারনেট)
   - Mobile Banking (মোবাইল ব্যাংকিং)
