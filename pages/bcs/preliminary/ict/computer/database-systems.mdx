# Database Systems (ডেটাবেস সিস্টেম)

## Key Topics (প্রধান আলোচ্য বিষয়)

1. **Database Concepts (ডেটাবেস ধারণা)**

   - Data vs Information (ডেটা বনাম ইনফরমেশন)
   - Database Management System (ডেটাবেস ম্যানেজমেন্ট সিস্টেম)
   - Database Models (ডেটাবেস মডেল)

2. **Types of Database (ডেটাবেসের প্রকারভেদ)**

   - Relational Database (রিলেশনাল ডেটাবেস)
   - Object-Oriented Database (অবজেক্ট-ওরিয়েন্টেড ডেটাবেস)
   - NoSQL Database (নোএসকিউএল ডেটাবেস)

3. **Database Components (ডেটাবেসের উপাদান)**

   - Tables (টেবিল)
   - Fields (ফিল্ড)
   - Records (রেকর্ড)
   - Keys (কী)

4. **Database Operations (ডেটাবেস অপারেশন)**
   - CRUD Operations (ক্রাড অপারেশন)
   - Queries (কোয়েরি)
   - Indexing (ইনডেক্সিং)
