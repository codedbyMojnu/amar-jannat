# 🌿 Amar Jannat — মজনু মিয়ার জান্নাত 🇧🇩

**Amar Jannat** হলো মজনু মিয়ার জ্ঞানভিত্তিক কল্পনার এক জান্নাত —  
তার **শান্তির জায়গা**, **সুখের জগৎ**, ও **নিজের পৃথিবী**।  
এখানেই তিনি তাঁর সারাজীবনের অর্জিত অভিজ্ঞতা, শিক্ষা, এবং বাস্তব জীবনের জ্ঞানের আলো ছড়িয়ে দিতে চান।

এই প্ল্যাটফর্মে জ্ঞান ও শিক্ষার এমন এক সমন্বয় থাকবে,  
যা বাংলাদেশের তরুণ প্রজন্মকে অনুপ্রাণিত করবে নিজের উন্নয়নের পথে হাঁটতে।

📘 **লাইভ সাইট:** [amarjannat.vercel.app](https://amarjannat.vercel.app)

## 🌍 উদ্দেশ্য ও দর্শন

> “জ্ঞানই জান্নাত — আর এই জান্নাত সবার জন্য উন্মুক্ত।”

**Amar Jannat** কেবল একটি ওয়েবসাইট নয়, এটি এক আজীবন শেখার ভ্রমণ।  
এখানে ধর্ম, বিজ্ঞান, প্রযুক্তি, এবং জীবনের অভিজ্ঞতা মিলেমিশে তৈরি করে এক ভারসাম্যপূর্ণ জ্ঞানের জগৎ।

🎯 **এই প্রকল্পের উদ্দেশ্য:**

1. বাংলায় মানসম্মত টেকনোলজি ও একাডেমিক কনটেন্ট তৈরি করা
2. বিসিএস ও পেশাগত জীবনের প্রস্তুতিতে সহায়তা করা
3. তরুণ প্রজন্মের জন্য সহজ ও আত্মার ছোঁয়াযুক্ত শেখার পরিবেশ তৈরি করা

নিচের ধাপগুলো অনুসরণ করুন 👇

1. **ফর্ক করুন** এই রিপোজিটরিটি।
2. **ক্লোন করুন** আপনার ফর্ক করা রিপোজিটরিটি:

```bash
   git clone https://github.com/codedbymojnu/amar-jannat.git
```

3. **নতুন ব্রাঞ্চ তৈরি করুন**:

   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **আপনার পরিবর্তন করুন** এবং **কমিট করুন**:

   ```bash
   git commit -m "আপনার পরিবর্তনের সংক্ষিপ্ত বিবরণ"
   ```

5. **পুশ করুন** আপনার ব্রাঞ্চ:

   ```bash
   git push origin feature/your-feature-name
   ```

6. **Pull Request** তৈরি করুন মূল রিপোজিটরিতে।
