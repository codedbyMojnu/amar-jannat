# Ditransitive Verb: Indirect and Direct Object

**প্রশ্ন:** He gave \_\_\_\_.

(a) to the class a tough assignment
(b) a tough assignment for the class
(c) the class a tough assignment
(d) an assignment tough to the class

**সঠিক উত্তর:** (c) the class a tough assignment

---

### **ব্যাখ্যা**

এই প্রশ্নটি Ditransitive Verb (যে verb দুটি object গ্রহণ করে) এর সঠিক বাক্য গঠন পরীক্ষা করার জন্য ডিজাইন করা হয়েছে। `give` একটি ditransitive verb।

#### **Key Concepts:**

1.  **Direct Object (DO):** Verb-কে 'কী' (what) দিয়ে প্রশ্ন করলে যে উত্তর পাওয়া যায়।
    - He gave **what**? -> a tough assignment (এটি Direct Object)।
2.  **Indirect Object (IO):** Verb-কে 'কাকে' (to whom) বা 'কার জন্য' (for whom) দিয়ে প্রশ্ন করলে যে উত্তর পাওয়া যায়।
    - He gave the assignment **to whom**? -> the class (এটি Indirect Object)।

#### **Ditransitive Verb-এর দুটি সাধারণ গঠন:**

**Structure 1: `Subject + Verb + Indirect Object + Direct Object`**

- এই গঠনে কোনো preposition (যেমন `to` বা `for`) বসে না।
- **উদাহরণ:** He gave **me** (IO) **a book** (DO).

**Structure 2: `Subject + Verb + Direct Object + Preposition + Indirect Object`**

- এই গঠনে Direct Object-এর পরে `to` বা `for` preposition বসে।
- **উদাহরণ:** He gave **a book** (DO) **to me** (IO).

#### **অপশনগুলোর বিশ্লেষণ:**

- **(a) to the class a tough assignment**: ভুল গঠন। Prepositional phrase (`to the class`) সাধারণত Direct Object-এর পরে বসে।
- **(b) a tough assignment for the class**: এটি ব্যাকরণগতভাবে সঠিক (Structure 2)। তবে `give` verb-এর সাথে সাধারণত `to` preposition ব্যবহৃত হয়, `for` নয়। (e.g., gave it **to** the class)।
- **(c) the class a tough assignment**: এটি Structure 1 (`IO + DO`) অনুসরণ করে এবং সম্পূর্ণ সঠিক। এটি সবচেয়ে সাধারণ এবং স্বাভাবিক গঠন।
- **(d) an assignment tough to the class**: ভুল শব্দক্রম। Adjective `tough` Noun `assignment`-এর আগে বসবে (`a tough assignment`)।

#### **উপসংহার:**

অপশন (c) হলো ditransitive verb `give`-এর সবচেয়ে সঠিক এবং প্রচলিত ব্যবহার। এখানে `the class` হলো Indirect Object এবং `a tough assignment` হলো Direct Object।

<Callout type="info">
  **Hack to Memorize:** মনে রাখবেন, ব্যক্তি (Indirect Object) যদি আগে আসে, তাহলে
  preposition লাগে না। বস্তু (Direct Object) যদি আগে আসে, তাহলে ব্যক্তির আগে
  `to` বা `for` preposition লাগে। * He gave **the class** (person) **the
  assignment** (thing). -> No preposition. * He gave **the assignment** (thing)
  **to the class** (person). -> Preposition needed.
</Callout>
