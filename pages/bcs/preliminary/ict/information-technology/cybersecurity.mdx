# Cybersecurity (তথ্য প্রযুক্তির ব্যবহার ও সাইবার নিরাপত্তা)

## Key Topics (প্রধান আলোচ্য বিষয়)

1. **Information Security (তথ্য নিরাপত্তা)**

   - Confidentiality (গোপনীয়তা)
   - Integrity (সত্যতা)
   - Availability (প্রাপ্যতা)
   - Authentication (প্রমাণীকরণ)

2. **Cyber Threats (সাইবার হুমকি)**

   - Malware (ম্যালওয়্যার)
   - Viruses (ভাইরাস)
   - Phishing (ফিশিং)
   - Ransomware (র্যানসামওয়্যার)

3. **Security Measures (নিরাপত্তা ব্যবস্থা)**

   - Firewalls (ফায়ারওয়াল)
   - Antivirus (অ্যান্টিভাইরাস)
   - Encryption (এনক্রিপশন)
   - Backup (ব্যাকআপ)

4. **Digital Safety (ডিজিটাল নিরাপত্তা)**
   - Password Security (পাসওয়ার্ড নিরাপত্তা)
   - Safe Browsing (নিরাপদ ব্রাউজিং)
   - Data Protection (ডেটা সুরক্ষা)
   - Privacy Settings (গোপনীয়তা সেটিংস)
