# Computer Programming and Software (কম্পিউটারের প্রোগ্রামিং ও সফটওয়্যার)

## Key Topics (প্রধান আলোচ্য বিষয়)

1. **Programming Concepts (প্রোগ্রামিং ধারণা)**

   - Algorithm (অ্যালগরিদম)
   - Flowchart (ফ্লোচার্ট)
   - Pseudocode (সুডোকোড)

2. **Programming Languages (প্রোগ্রামিং ভাষা)**

   - Machine Language (মেশিন ল্যাংগুয়েজ)
   - Assembly Language (অ্যাসেম্বলি ল্যাংগুয়েজ)
   - High-Level Languages (হাই-লেভেল ল্যাংগুয়েজ)

3. **Types of Software (সফটওয়্যারের প্রকারভেদ)**

   - System Software (সিস্টেম সফটওয়্যার)
   - Application Software (অ্যাপ্লিকেশন সফটওয়্যার)
   - Utility Software (ইউটিলিটি সফটওয়্যার)

4. **Operating Systems (অপারেটিং সিস্টেম)**
   - Windows (উইন্ডোজ)
   - Linux (লিনাক্স)
   - macOS (ম্যাক ওএস)
