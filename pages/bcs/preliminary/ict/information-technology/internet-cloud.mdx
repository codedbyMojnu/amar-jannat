# Internet and Cloud Computing (ইন্টারনেট ও ক্লাউড কম্পিউটিং)

## Key Topics (প্রধান আলোচ্য বিষয়)

1. **Internet Basics (ইন্টারনেট মৌলিক বিষয়)**

   - Internet Protocol (ইন্টারনেট প্রোটোকল)
   - Domain Name System (ডোমেইন নেম সিস্টেম)
   - Web Browsers (ওয়েব ব্রাউজার)
   - Search Engines (সার্চ ইঞ্জিন)

2. **World Wide Web (ওয়ার্ল্ড ওয়াইড ওয়েব)**

   - Web Pages (ওয়েব পেজ)
   - Web Servers (ওয়েব সার্ভার)
   - Web Hosting (ওয়েব হোস্টিং)
   - Web Security (ওয়েব সিকিউরিটি)

3. **Cloud Computing (ক্লাউড কম্পিউটিং)**

   - Cloud Services (ক্লাউড সেবা)
     - IaaS
     - PaaS
     - SaaS
   - Cloud Storage (ক্লাউড স্টোরেজ)
   - Cloud Security (ক্লাউড সিকিউরিটি)

4. **Internet Services (ইন্টারনেট সেবা)**
   - Email (ইমেইল)
   - VoIP (ভিওআইপি)
   - File Sharing (ফাইল শেয়ারিং)
   - Video Streaming (ভিডিও স্ট্রিমিং)
