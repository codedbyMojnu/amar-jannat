# Computer Networks (কম্পিউটার নেটওয়ার্ক)

## Key Topics (প্রধান আলোচ্য বিষয়)

1. **Types of Networks (নেটওয়ার্কের প্রকারভেদ)**

   - LAN (ল্যান)
   - MAN (ম্যান)
   - WAN (ওয়ান)
   - PAN (প্যান)

2. **Network Topologies (নেটওয়ার্ক টপোলজি)**

   - Bus (বাস)
   - Star (স্টার)
   - Ring (রিং)
   - Mesh (মেশ)

3. **Network Protocols (নেটওয়ার্ক প্রোটোকল)**

   - TCP/IP (টিসিপি/আইপি)
   - HTTP (এইচটিটিপি)
   - FTP (এফটিপি)
   - SMTP (এসএমটিপি)

4. **Network Security (নেটওয়ার্ক সিকিউরিটি)**
   - Firewall (ফায়ারওয়াল)
   - Encryption (এনক্রিপশন)
   - VPN (ভিপিএন)
